Open Next Router (ONR) is a routing and transformation runtime for provider APIs.

## Key Ideas

- Runtime behavior is declared explicitly with DSL directives.
- The proxy layer is an execution engine, not a rule-guessing layer.
- Compatibility and transformations are opt-in through directives.
